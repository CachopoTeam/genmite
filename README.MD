# Code generator

## Install

```sh
npm install
```

## Run Examples

1.- Build the project
```sh
  npm run build
```

2.- Create a folder demo

3- Run a npm script like

```sh
  npm run generate:component:angular
```

4.- Enter as Destination folder "demo" and as component folder hello.
